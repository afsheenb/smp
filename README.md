# Something More Profound

A book about the political economy of network protocols, built with Pandoc.

## Compiling

Requirements:

- [Pandoc](https://pandoc.org/)
- [Just](https://github.com/casey/just)

Build all formats:

```
just
```

Build a single format:

```
just html
just epub
just pdf
```

Delete built files:

```
just clean
```
